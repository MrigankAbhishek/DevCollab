# DevCollab 🚀

A real-time collaborative coding platform where developers can write code together, chat, manage tasks, and share files — all in one room.

🔗 **Live Demo**: [dev-collab-rouge.vercel.app](https://dev-collab-rouge.vercel.app)  
📁 **GitHub**: [github.com/MrigankAbhishek/DevCollab](https://github.com/MrigankAbhishek/DevCollab)

---

## Overview

DevCollab lets developers collaborate on code in real-time — like Google Docs but for code. Create a room, share the invite code, and your teammates can join instantly. Every keystroke is synced live, with a built-in chat and Kanban board alongside the editor.

## Features

- ⚡ Real-time collaborative code editor with live sync across all users
- 💬 Per-room live chat
- 📋 Drag-and-drop Kanban board (To Do → In Progress → Done)
- 🔒 GitHub OAuth login with JWT authentication
- 📁 File and snippet uploads via Cloudinary
- 🌐 12+ programming languages supported
- 👥 Online presence indicators
- 🔗 Room invite codes for easy access
